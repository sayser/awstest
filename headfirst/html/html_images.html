HTML Images

HTML images are defined with the <img>

<img> tag is a void element, it contains attributes only, and does not have a closing tag.
The src attribute specifies the URL of the image

<img src="url" alt="some_text">

Code

<img src="../images/lucaliya001.jpg" alt="Picture JPG" style="width:300px; height:350px;">

Image Floating

Use the CSS float property to let the image float.
The image can float to the right or the the left of a text:

Code

<img src="../images/lucaliya001.jpg" alt="Picture JPG" style="float:left;width15px;height:20px;">

<img src="../images/lucaliya001.jpg" alt="Picture JPG" style="float:right;width15px;height:20px;">
