HTML Links

HTML links are hyperlinks, A hyperlink is a text or an image you can click on, and jump to another document In HTML, links are defined with the <a>

Syntax

<a href="url">link text</a>

Examples

External Link

<a href="http://somesitme.com">Click to Visit External Site</a>

Local Link

<a href="somefolder/somefile.html">Click to Visit Internal html file</a>

Image Link

<a href="http://www.google.com"><img src="../images/googlelogo.png"></a>

Creating Bookmarks

HTML Bookmarks are used to allow readers to jump to specific parts of a web page

First Create the bookmark with "id" attribute:
<h2 id="tips">Useful Tips Section</h2>

Then, add a link to the bookmark:
<a href="html_tips.html#tips">Visit the Useful Tips Section</a>

Attributes

The target attribute specifies where to open the linked document
_blank Opens the linked document in a new window or tab
<a href="somepage.html" target="_blank">Open link in new window</a>
